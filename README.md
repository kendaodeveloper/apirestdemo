# API REST Demonstration

Demonstração de uma API REST.

# Requisitos

- Java 11

# Comandos

Compilar o projeto:

```sh
./gradlew build -x test
```

Rodar os testes:

```sh
./gradlew test
```

Executar o projeto:

```sh
./gradlew bootRun
```
